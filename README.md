## 4. Introducción

Este proyecto consiste en la implementación de 10 problemas de programación en PHP que demuestran el uso de estructuras de control, funciones y clases. Cada problema aborda diferentes conceptos fundamentales de programación aplicados en contextos reales como cálculos estadísticos, procesamiento de datos, y sistemas de gestión.

## 5. Características Técnicas

### Programación Orientada a Objetos (POO)
El proyecto implementa principios de POO a través de:

- **Clase Utilities**: Contiene métodos estáticos reutilizables
- **Encapsulamiento**: Separación de lógica y presentación
- **Reutilización**: Funciones modulares y componentes compartidos
